Chat App Design Writeup

Jonathan Zhao

Actions:

1. Create Chat

What: Post a chat Graffiti object with a user-specified title and random channel UUID.

Example Object JSON:

{ "value": { "activity": "Create", "type": "Chat", "title": "6.4500 Study Group", "channel": "d222491b-d6d0-4c21-aadd-45ed9bc21566", "published": 1777065600000 }, "channels": ["designftw-26"] }

Where: Posted to the shared directory channel "designftw-26", so new chats are discoverable by everyone. subject to change in the future with actual API and security features.

Who: Currently anyone in the shared directory channel "designftw-26" can create a chat. Anyone in "designftw-26" can see new chats.

2. Join Chat

What: Post a join Graffiti object with an existing, user-specified chat channel UUID.

Example Object JSON:

{ "value": { "activity": "Join", "target": "d222491b-d6d0-4c21-aadd-45ed9bc21566", "published": 1777065605000 }, "channels": ["d222491b-d6d0-4c21-aadd-45ed9bc21566"] }

Where: Posted to the specified channel UUID, so membership is tied to that specific chat room.

Who: Currently anyone in the shared directory channel "designftw-26" can join any chat, visible from the chat list on the left.

3. Send Message

What: Post a message Graffiti object with a content field and published timestamp.

Example Object JSON:

{ "value": { "content": "Can someone explain problem 3?", "published": 1777065610000 }, "channels": ["d222491b-d6d0-4c21-aadd-45ed9bc21566"] }

Where: Directly in the current chat channel, so messages are visible to everyone in the chat. Discover messages from the same channel using schema filtering.

Who: Public to everyone in the current chat channel.

4. Pin Message

What: Post a pin Graffiti object with a target message URL field, content field for the message, timestamp.

Example Object JSON:

{ "value": { "content": "Can someone explain problem 3?", "published": 1777065610000 }, "channels": ["d222491b-d6d0-4c21-aadd-45ed9bc21566"] }

Where: Posted to the current chat channel. Discovered from the same channel and shown in the pinned section on the right.

Who: Public in the chat channel. Pinning does not make the message private in any way.
